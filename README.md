# Vue3茶叶后台管理系统 java121

## 项目概述

本项目是一款基于Vue3的纯前端茶叶后台管理系统，采用模块化设计，无数据库依赖，提供完整的开发文档，支持定制开发。

## 技术栈

- **前端框架：** Vue3
- **UI组件库：** Element Plus
- **构建工具：** Vite
- **状态管理：** Pinia
- **路由管理：** Vue Router

## 功能模块

系统包含以下6个页面：

1. 登录页
2. 后台布局页
3. 茶叶分类管理
4. 茶叶商品管理
5. 个人资料页
6. 404错误页面

### 自定义组件

- 茶叶详情对话框组件
- 头像上传组件

### 功能特性

- 分页功能
- 表格操作
- 头像上传功能
- 父子组件通信（prop/emit）
- 指令支持（v-if/v-for/v-on/v-model/v-bind）
- 计算属性（computed）与侦听器（watch）

## 系统角色

- 系统管理员：负责系统的日常管理和操作

## 运行环境

- 现代浏览器

## 部署步骤

1. 克隆或下载项目代码
2. 安装依赖（运行`npm install`或`yarn`）
3. 运行项目（运行`npm run dev`或`yarn dev`）
